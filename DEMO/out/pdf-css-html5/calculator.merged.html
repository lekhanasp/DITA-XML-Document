How to Start the Calculator App

Here you will learn how to start the Calculator

To use the calculator app, you need to unlock your phone first.

You can use calculator app on your smartphone. You will find it useful, when you need to quickly calculate some numbers.

  1. Navigate to the Calculator application icon
  2. Tap the icon

The application starts.

You are now ready to calculate with it.

For example, if you have purchased a book for Rs.15 and you have given Rs.50, how much change should you expect? Answer is 50-15 = 35
